# AssistantTextBot
# Телеграм бот, который помогает с текстом
Этот бот позволяет посчитать количество символов в тексте или посчитать сумму чисел, написанных через пробел.
## Использование
Бота можно подключать в чаты и переписыватся с ним в личке.
Можно использовать его как есть.
Если зарегистрируете своего, то замените BotToken в классе Program.cs в строчке BotToken = "8541085617:AAG_Tm2xBlrd4CJOXLDeMNyi8EdQTROd0Ug", своим значением.
